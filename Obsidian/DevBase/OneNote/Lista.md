- C# Sintaxe Básica + Avançada:

- Foque em LINQ, Async/Await e Generics. Sem isso, o código .NET parece grego.

- ​.NET Web API: Aprenda a criar uma API simples, conectar no banco (Entity Framework Core) e devolver um JSON.

- ​Next.js Básico: Crie o front que consome esse JSON da sua API.

- ​DDD: Só depois de ter o fluxo "Front <-> Back <-> Banco" funcionando, comece a refatorar o back-end aplicando DDD para organizar a casa.
